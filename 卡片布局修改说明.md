# AOI 质量看板 — 卡片布局修改说明

## 修改目标
统一所有卡片标题栏控件位置，将散落在卡片内容区的按钮/下拉框迁移到标题栏右侧，实现一致的对齐效果。

---

## 1. 控件插入位置调整

**文件：** `dashboard/core/dashboard_card.py`

**改动：** `add_header_widget` 方法中，插入索引从 `hl.count() - 2` 改为 `hl.count() - 1`

```
- hl.insertWidget(hl.count() - 2, widget)  # 旧：stretch 之前
+ hl.insertWidget(hl.count() - 1, widget)  # 新：stretch 之后，max_btn 之前
```

**效果：** 所有附加控件从标题栏左侧移至右侧，布局统一为：

```
[标题][stretch......][附加控件...][🗖]
```

---

## 2. 热力图切换按钮

**文件：** `dashboard/pages/home_page.py`

**改动：** `_build_ui()` 中，将 `SchematicHeatmapCard` 内部的 `_toggle_btn` 移至 `DashboardCard` 标题栏：

```python
self._heatmap_card.add_header_widget(self._heatmap_widget._toggle_btn)
self._heatmap_widget.layout().takeAt(0)  # 移除原内部空布局
```

---

## 3. 记录筛选按钮

**文件：** `dashboard/pages/home_page.py`

**改动：** `_build_ui()` 中，将 `RecentRecordsTable` 内部的三个筛选按钮（NG/OK/全部）移至 `DashboardCard` 标题栏：

```python
for btn in self._records_table._filter_buttons.values():
    self._records_card.add_header_widget(btn)
self._records_table.layout().takeAt(0)  # 移除原内部空筛选行
```

---

## 4. 图片查看器状态栏

**文件：** `dashboard/cards/image_viewer_card.py`

**改动：** `_build_ui()` 中，`status_label` 不再放入 content 布局，改为加入标题栏右侧：

```python
- layout.addWidget(self.status_label)          # 旧：内容区顶部
+ self.add_header_widget(self.status_label)    # 新：标题栏右侧
```

**效果：** 右侧 `name_list` 向上延伸一行高度，空间利用更充分。

---

## 最终布局对照

| 卡片 | 标题栏控件（从右到左） |
|------|------------------------|
| 良率趋势 | `[🗖]` ← 下拉框 |
| 缺陷分布 TOP10 | `[🗖]` ← 柱状图切换按钮 ← 下拉框 |
| Pareto 分析 | `[🗖]` ← 下拉框 |
| 最近检测记录 | `[🗖]` ← 全部 ← OK ← NG |
| 区域热力图 | `[🗖]` ← 区域视图切换按钮 |
| 图片查看器 | `[🗖]` ← 当前筛选信息 |

所有卡片标题栏控件**右对齐**，排列**顺序一致**。
